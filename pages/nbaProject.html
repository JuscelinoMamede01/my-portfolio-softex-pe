La Basketeira

Project of a basketball jersey registration e-commerce page, where the retailer registers the product, and upon registering the jersey, it is automatically displayed on the screen with all the information.

Project Overview

The objective of the "La Basketeira" project is to provide NBA fans with a convenient way to purchase jerseys from their favorite teams while offering an engaging and informative user experience.This is a project summary based on the provided HTML code. You can further expand and customize the project by adding additional features such as a shopping cart system, product detail pages, and an online payment system.

Tools used

HTML CSS PHP MySQL Xampp
